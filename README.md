# MyMovieGuide
电影手册项目，使用MVP, RxJava, OkHttp, Retrofit, Dagger2等进行架构设计。

Add your [themoviedb.org](https://www.themoviedb.org/) API key in the `local.properties` file:
```
tmdb_api_key=YOUR_API_KEY
```
